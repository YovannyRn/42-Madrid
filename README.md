## 🏊 ¿Qué es la Piscina de 42?

La **Piscina** es el proceso de selección de 42 Madrid: 26 días intensivos de programación en C y Shell donde se evalúa no solo tu nivel técnico, sino tu capacidad de aprendizaje, resiliencia y trabajo en equipo. Sin profesores, sin clases magistrales — solo tú, los recursos y los demás cadetes.
